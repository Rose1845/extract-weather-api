# extract-weather-api

## Weather API ETL Pipeline

### Overview

This project implements a complete ETL (Extract, Transform, Load) pipeline using Python to fetch weather forecast data from an external API, process it, and store it in a PostgreSQL database.

The pipeline demonstrates core data engineering concepts including:

- API data extraction
- Data transformation and cleaning
- Database loading
- Environment-based configuration

#### Tech Stack

Python
Requests (API calls)
PostgreSQL
psycopg2
python-dotenv
